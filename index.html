Projects

Data Structures

In this course I built 3 Java Applets. All were from scratch except for the KdTree applet, which my professor provided some of the helper classes for. Seeing as how Applets are essentially obsolete, I working on translating them to JavaScript so that I may embed them here...For now I'll leave the code and some demonstration videos.



Applets:

  • Postfix Calcuator

    • This calculator employs postfix notation. Press a number and "enter" to push it onto the stack. Pressing an operation key will push the two top values from the stack and perform the operation. Press pop to remove the mose recent value entered.

  • K-d Tree Builder

    • This program allows the user to build a K-dimensional tree, a binary search tree where the data in each node is a point in space. Press anywhere to add a point to the tree, press the "rebuild" button to balance it.

  • Game of Memory

    • This is a game my friend Halcyon and I made at the end of the semester. It's a simple Memory game where the user selects two tiles and if they are the same color the tiles disappear and the user wins once they've guessed them all. It can be played in 4x4 or 6x6 and there is a timer. Once I've converted it to a program that can be embedded on the web I'm going to switch the colored squares for photos of dogs, and the game will be to match the dogs of the same breed!
